# example
```
MATCH (p:Person)-[:WORKS_FOR]->(c:Company)
WHERE c.name STARTS WITH "Face"
AND EXISTS {
  MATCH (p)-[:LIKES]->(t:Technology)
  WHERE size((t)<-[:LIKES]-()) >= 3
}
RETURN p.name as p, c.name AS company;
```
